## Description

BlackArch Linux configuration files for gtk2.

## Installation

`pacman -S blackarch-config-gtk`

## Configuration

Copy the following files:

1. `cp -a /usr/share/blackarch/config/gtk-2.0/* /usr/share/gtk-2.0/`
